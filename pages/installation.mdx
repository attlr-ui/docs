## Installation

### 1. Install @attlr/ui-components

```sh copy npm2yarn 
npm install @attlr-ui/ui-components
```


### 2. Install @attlr/cli (Recommended)

Attlr CLI provides a convenient way to install only the components you need.

```sh copy npm2yarn
npm install -D @attlr/cli
```

## Usage

### Importing Components

Once installed, you can import components from `@attlr-ui/ui-components` in your React Native application:

```tsx copy
import { AButton, AInput } from '@attlr-ui/ui-components';
```

### Using Attlr CLI (Recommended)

If you've installed Attlr CLI, you can use it to install specific components:

```sh copy  /add/
npx attlr add abutton 
```
#### Aliases
- `a`
- `install`
